# has-pkg

[![NPM version](https://img.shields.io/npm/v/has-pkg?color=a1b858&label=)](https://www.npmjs.com/package/has-pkg)

Check if a package is installed locally.

```ts
import hasPkg from 'has-pkg'

hasPkg('lodash') // true
hasPkg('@types/lodash') // false
```
